## How choose command? 
* {__} - your data

| Command |   Description   |
| --- | :---:
| `node index.js -a list`   | Show all contacts    |
| `node index.js -a get -i {id}`  | Get contact by Id  |
| `node index.js -a add -n {YourName} -e {YourEmail} -p {YourPhone}`   | Add contact |
| `node index.js -a remove -i {id}`  |Remove contact |


| Short |   Full  |
| :---: | :---:
|  `-i`   | `--id`    |
|  `-n`   | `--name`  |
|  `-e`   | `--email` |
|  `-p`   | `--phone` |
